## How to Compile

### Requirements
- [Node.js](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

### Install
`yarn`

### Run
`yarn start`  
You can then open it up on a web browser to http://127.0.0.1:8080

### Build Prod
`yarn build:prod`  
Files will be in the `public` folder
